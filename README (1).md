# TechMarket — завершене вебрішення

Навчальний вебпроєкт інтернет-магазину техніки.

## Структура

- `index.html` — головна сторінка;
- `catalog.html` — каталог товарів із фільтрацією;
- `contacts.html` — контакти, форма, iframe-карта, медіаблок;
- `register.html` — сторінка реєстрації;
- `grid.html` — демонстрація CSS Grid;
- `bootstrap.html` — демонстрація Bootstrap-сітки;
- `css/style.css` — стилі сайту;
- `js/script.js` — JavaScript для фільтрації та форм.

## Як запустити

1. Розархівувати папку.
2. Відкрити файл `index.html` у браузері.
3. Для GitHub Pages завантажити всі файли в репозиторій і увімкнути Pages для гілки `main`.
