# TechStore — завершене вебрішення

## 1. Загальна інформація

**TechStore** — це завершене навчальне вебрішення у вигляді інтернет-магазину техніки.

Сайт створений як багатосторінковий вебпроєкт із головною сторінкою, каталогом товарів, сторінкою контактів, пошуком, фільтрами, картками товарів, зображеннями, контактною формою, iframe, аудіо- та відеоблоком.

Проєкт не використовує серверну частину та базу даних. Уся логіка реалізована на стороні клієнта за допомогою HTML, CSS та JavaScript.

## 19. Запуск через Visual Studio Code

Сайт також можна запускати через Visual Studio Code.

Порядок запуску:

1. Відкрити VS Code.

3. Встановити розширення **Live Server**.
4. Натиснути правою кнопкою миші на `index.html`.
5. Обрати **Open with Live Server**.
6. Сайт відкриється у браузері.

Такий спосіб зручний під час редагування сайту.

## 20. Публікація на GitHub Pages

Щоб опублікувати сайт на GitHub Pages, потрібно:

1. Створити репозиторій на GitHub.
