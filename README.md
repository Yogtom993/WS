# TechStore — великий інтернет-магазин

## Опис
Завершений сайт TechStore у форматі інтернет-магазину техніки.

## Каталог
У каталозі додано **337 товарів**. Кількість товарів у категоріях спеціально нерівна, щоб каталог виглядав природніше:

- Ноутбуки — 64 товарів;
- Смартфони — 58 товарів;
- Аудіо — 47 товарів;
- Аксесуари — 73 товарів;
- Монітори — 39 товарів;
- Накопичувачі — 56 товарів;

## Що працює
- категорії товарів;
- фільтри;
- пошук;
- зображення товарів;
- форма заявки;
- фрейм;
- аудіо та відео.

## Файли
- `index.html` — головна сторінка;
- `catalog.html` — каталог;
- `contacts.html` — контакти й заявка;
- `style.css` — стилі;
- `script.js` — генерація товарів, пошук і фільтри;
- `images/` — зображення товарів;
- `media/` — аудіо та відео.

## Запуск
Відкрити `index.html` у браузері.
